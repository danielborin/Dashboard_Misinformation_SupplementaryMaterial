# Dashboard_Misinformation_SupplementaryMaterial
This repository contains the supplementary material for the paper **Thirty Years of Misinformation Research: A team science perspective (1994-2024)"** authored by **Daniel Borin**, **Diego Fregolent Mendes de Oliveira** and **Brian Uzzi**. The project offers an interactive Streamlit dashboard that allows users to explore several features of the analysis of misinformation research.

## Live Dashboard

You can view the interactive dashboard here:  

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://misinformation.streamlit.app)

## Running the Dashboard Locally

To run this Streamlit dashboard locally, follow these steps:

### 1. Clone the repository:

```
git clone https://github.com/danielborin/Dashboard_Misinformation_SupplementaryMaterial.git
cd Dashboard_Misinformation_SupplementaryMaterial
```

### 2. Set up a virtual environment (optional but recommended):

```
virtualenv .venv
source .venv/bin/activate
```

### 3. Install the required dependencies:

```
pip install -r requirements.txt
```

### 4. Run the dashboard:

```
streamlit run dashboard_misinformation.py
```

This will launch the Streamlit app in your default web browser.

## Requirements

To run the dashboard locally, you'll need the Python libraries which are listed in the `requirements.txt` file and can be installed using `pip`.
